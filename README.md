# Electricity Billing System

This is a Java-based desktop application developed using Swing and JDBC.

## Technologies Used
- Java (Swing)
- JDBC
- MySQL
- NetBeans

## Features
- Customer management
- Bill generation
- Payment tracking

## How to Run
- Open project in NetBeans
- Connect to MySQL database
- Run the project

## Note
This project demonstrates basic DBMS and Java GUI concepts.
